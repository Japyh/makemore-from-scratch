# makemore-from-scratch

A from-scratch reimplementation of the ideas behind Andrej Karpathy’s **makemore**, built for learning, readability, and experimentation.

This project explores character-level language modeling step by step, starting from simple statistical baselines and moving toward neural network models. The focus is on understanding how the models work internally, writing the code myself, and improving the developer experience with cleaner structure, better comments, and easier experimentation.

## What this repo includes

- character-level data pipeline
- baseline models and neural models
- training and sampling code
- clearer organization than a quick prototype
- notes and experiments for learning

## Why this repo exists

I built this repository to understand generative language models more deeply by recreating the ideas from scratch instead of only reading them. It is inspired by Karpathy’s original makemore project, but rewritten in my own style with incremental improvements.

## Goals

- learn by implementing
- keep the code compact and understandable
- make experimentation easier
- document insights while building
